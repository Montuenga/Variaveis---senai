# JavaScript - Exercícios básicos

Repositório com arquivos simples para praticar conceitos básicos de JavaScript.

## Arquivos

* **armazenamento.js** – prática de armazenamento de dados em variáveis.
* **declaração.js** – exemplos de declaração de variáveis.
* **esboço.js** – testes de lógica básica, funções e condições.

## Objetivo

Praticar fundamentos da linguagem JavaScript e entender melhor o funcionamento de variáveis, funções e estruturas condicionais.
